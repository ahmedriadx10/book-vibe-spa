# BOOK VIBE



**Book Vibe** is a clean and modern React Single Page Application (SPA) that helps book lovers manage their reading journey effortlessly.

Users can easily add books to **Readlist** or **Wishlist** with a smart toggle system — adding a book to one list automatically removes it from the other. All data is persistently saved in **localStorage**, ensuring your lists remain safe even after refreshing the browser.

Built with **React Router**, the app features nested routing and dynamic routing for smooth navigation and individual book detail pages. 

A standout feature is the **reading analytics chart** that visually shows which book in your Readlist has received the most page visits.

## ✨ Features

- **Smart Toggle System**  
  Adding a book to Readlist automatically removes it from Wishlist and vice versa.

- **LocalStorage Persistence**  
  Readlist and Wishlist data are automatically saved in the browser.

- **React Router Implementation**  
  Nested routing and dynamic routing for seamless navigation and book details pages.

- **Reading Analytics**  
  Interactive chart on Readlist book detail pages showing which book has the highest page visits.

- **Responsive & Modern Design**  
  Clean, vibrant, and fully mobile-friendly user interface.

- **Fast SPA Experience**  
  Built with React for smooth and fast performance.
